# Baconfy UI

![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/baconfy/ui/testing.yml?branch=main&style=plastic)

## Description

Baconfy UI is the user interface package of the Baconfy ecosystem. It provides reusable UI components and tools to accelerate the development of modern, responsive web applications. This package is designed to seamlessly integrate with other Baconfy ecosystem components, offering developers a cohesive and efficient workflow.
